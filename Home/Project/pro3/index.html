Automatic Blending Machine

หลักการทำงาน

ใช้ปั๊มลมในการดันน้ำขึ้นมาผสมควบคุมโดยโปรแกรมใน Arduino

วิธีการใช้งาน

เมื่อกดปุ่มปั๊มน้ำจะทำงานโดยปล่อยลมไปดันน้ำในภาชนะทั้งสองขึ้นมาในปริมาณที่ได้กำหนดสัดส่วนไว้ด้วยเวลาที่ได้ออกแบบไว้ในโปรแกรม โดยสามารถนำมาประยุกต์ใช้กับการชงเครื่องดื่ม หรือ การรดน้ำต้นไม้การทำสวนได้

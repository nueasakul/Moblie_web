Safety cap

หลักการทำงาน

เซนเซอร์ตรวจจับความเร่งแบบ 3 แกน (accelerometer sensor : รุ่น ADXL345) จะถูกติดตั้งไว้ที่ด้านบนบริเวณกึ่งกลางของหมวก เมื่อหมวกมีการเอียงไปมาตัวเซนเซอร์จะวัดค่าแกน X Y และ Z แล้วนำค่าที่ได้มาคำนวณเป็นองศาและเขียนโปรแกรมผ่าน Arduino เพื่อควบคุมหลอดไฟ LED ให้มืด-สว่างตามองศาที่กำหนด
